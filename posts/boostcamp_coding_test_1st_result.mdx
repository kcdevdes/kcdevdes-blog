## 😃 1차 합격

친구와 같이 밥을 먹다 친구가 메일을 받았다고 말했다. 개인적으로 _'~~나는 1.5솔이고 기도도 매일 드렸고 자소서도 잘 작성해서 1차는 붙여주겠지~~~'_ 라는 생각을 했지만 막상 메일이 순차적으로 온다는 것을 들으니 먹고 있던 국밥이 역류하는 기분이었다. 긴장되는 마음으로 메일을 기다리다 확인했더니 다행스럽게도 이후의 기회가 주어졌다.

오픈 채팅방에서도 합격하신 분들의 말이 이어졌는데 비전공자들 중에서 0솔 ~ 0.5솔 정도 하신 분들은 거의 탈락의 슬픔을 맛보신 것 같았다. ( 그렇다고 다 떨어지신건 아니다 ) 전공자들 중에서는 0솔도 합격 사례가 있는 것을 보면 **CS 지식을 잘 보셨거나 자소서를 잘 쓰신 것** 같다.
